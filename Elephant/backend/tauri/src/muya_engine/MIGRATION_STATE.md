Each module has Rust unit tests. Deterministic modules are additionally compared against fixtures generated by executing the JavaScript oracle. Observable JavaScript details such as UTF-16 lengths and offsets, truthiness, stringification, insertion order, escaped table pipes and exact HTML output are represented explicitly.

These modules compile through `markdown::muya_engine`. They are not yet the active frontend runtime: the first coherent parser slice must be completed and pass the full original-vs-Rust characterization gate before the corresponding JavaScript files are removed.
