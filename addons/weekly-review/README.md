# Weekly Review

An original periodic-notes workflow inspired by popular daily and weekly note plugins.

The addon reads the most recent dated notes matching `Daily/YYYY-MM-DD.md` and writes `Periodic/Weekly/YYYY-Www.md`.

The review contains:

- open tasks;
- completed tasks;
- bullet highlights;
- links back to every source daily note.

The default window is seven notes. Programmatic callers may pass `{ days }`, capped at fourteen.

The addon never edits daily notes.
