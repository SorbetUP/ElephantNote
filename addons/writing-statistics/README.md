# Writing Statistics

An original vault-wide writing report inspired by word-count and vault-statistics plugins.

It measures:

- total words and Unicode characters;
- estimated reading time at 200 words per minute;
- the twenty largest notes;
- activity grouped by modification date;
- change in words and note count since the previous run.

The report is written to `Reports/Writing Statistics.md`. The previous aggregate is kept in private addon storage; note bodies are not stored there.
