# Portage Blinko vers Elephant

Ce dossier contient les documents de travail pour reprendre les fonctionnalites ajoutees dans Blinko Offline et les porter dans l'app personnelle Elephant.

Fichiers:

- `CONVERSATIONS_CODEX_BLINKO.md`: historique resume des conversations Codex qui ont mene aux ajouts Blinko.
- `CHECKLIST_FONCTIONNALITES_BLINKO_A_PORTER.md`: checklist des fonctionnalites a reprendre, avec descriptions et reperes techniques.

Important: ces fichiers ne sont pas une documentation du fork Blinko en tant que cible finale. Ils servent de cahier de portage pour implementer les idees utiles dans Elephant avec l'architecture propre a ce projet.
