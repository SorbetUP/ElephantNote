## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-setcolor`
- `allow-open-app-settings`
- `allow-present-share-sheet`
- `allow-get-pending-share-payload`
- `allow-clear-pending-share-payload`
