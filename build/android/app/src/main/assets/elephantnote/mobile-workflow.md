# Mobile workflow

folder: Guides
#workflow #mobile #sync

Use this release as a capture-first vault that keeps the same concepts as
desktop ElephantNote while staying practical on Android.

## Capture

Start in Notes. A note can be as small as one line or can contain full Markdown:

```markdown
folder: Research
#source #idea

# Question

What did we learn from [[Project Atlas]]?
```

The first non-empty line becomes the title when no explicit title is supplied.
Tags are indexed immediately. A `folder:` line moves the note into a vault
folder without requiring a separate file picker.

Use the Markdown toolbar in quick capture or note editing to insert headings,
bold, italic, task items, wiki links, inline code, quotes, tags and folder
metadata without typing every marker manually.

Open a note from the list to see a native Markdown preview with rendered
headings, lists, tasks, quotes, code blocks and image references. Use Edit when
you need the raw Markdown source again.

## Organize

Vault shows folder groupings and note counts. Search scans the complete local
note body, titles, folders and tags. Graph summarizes relationships so the
mobile vault remains navigable even before desktop sync.

Wiki and Graph topics are clickable: open a tag topic to inspect the notes that
formed it, then tap any linked note for preview or editing.

## Enrich

Sources stores URLs and creates matching source notes. Calendar stores lightweight
events and recent note activity. Tasks can generate a briefing note from the
current local state: notes, words, tags, backlinks, calendar events and sources.

## Attach

Share one or more images to ElephantNote from Android. The app copies those
files into private local attachment storage and creates Markdown image links.
Use Files to review stored attachments, copy a file URI, generate a fresh
attachment note or delete files that are no longer needed.

## Draw

Canvas stores touch drawings locally. Each drawing is a compact stroke document
that can be reopened, edited and deleted from the Canvas tab.

## Handoff

Use Sync to configure the device identity and remote path expected by the
desktop sync setup. Use Copy Markdown export for a deterministic Markdown bundle
that can be pasted, saved or transferred.
